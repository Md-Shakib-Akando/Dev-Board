Question-1: What are the different ways to select an element in the DOM?

The main ways to select an element in the DOM, getElementById() getElementsByClassName() getElementsByTagName() querySelector() querySelectorAll().All are way to select an element in the DOM.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

"InnerHtml" returns all text, including html tags, that is contained by an element. The "InnerText" property sets or returns the text content of an element. "TextContent" property sets or returns the text content of the specified node, and all its descendants.

Question-3: What is event delegation in the DOM?

Event delegation is is basically a pattern to handle events efficiently where we can add event listener to a parent element to handle events from its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling directs an event to its target. When an element (like a button) is clicked, an event is directed to the element.where the event bubbles up from the target element all the way to the global window object.

Question-5: How do you create, add, and remove elements using JavaScript?

To create a new element, use document.createElement(). Add an Element use appendChild(). Remove an element, use removeChild().
